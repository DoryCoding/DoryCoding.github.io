github创建博客流程

参考网址：

简书（上篇）

简书（下篇）

个性化配置1

个性化配置2

配置好以后每次只要执行简单的命令就可直接更新官网的代码。步骤：

hexo new “文章名”

hexo clean

hexo generate （或者hexo g）

hexo deploy （或者hexo d）

如何想在本地查看效果hexo server （或者hexo s）

然后打开浏览器输入：localhost:4000即可查看

附：

每次hexo d之后重新刷新自己的网址，就会提示404 not found。进自己的项目仓库的setting看一下，Custom domain为空了。重新加上自己的域名就可以，但是不能每次都这样吧。

解决办法是在public文件夹下面新建一个CNAME文件（注意没有后缀），里面输入你的域名保存退出。下次就不会出现这种问题，解决！
