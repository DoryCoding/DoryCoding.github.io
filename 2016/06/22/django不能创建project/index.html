django不能创建project

本地的博客写好之后，部署到阿里云服务器上时，pip install django能成功，但是

django-admin startproject *老是不成功，并报以下错误：

stackoverflow上都是说服务器上有两个python环境或者两个django，执行两次

pip uninstall django就可以成功，但是对我没用。

最后还是暴力解决问题，进入python2.7的路径，在里面找到所有和django相关的package，

执行rm -rf *把相关的package全部删除，再执行pip install django就OK了！
